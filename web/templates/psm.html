Propensity Score Matching

Estimate causal effects from observational data by matching treated and control units on covariates.

When to use: You have observational data with no random assignment — users self-selected into treatment (e.g. opted into a feature, received a notification). PSM finds control units that are statistically similar to each treated unit based on covariates, then estimates the Average Treatment Effect (ATE) on the matched sample.

Data structure: All four columns must have the same number of rows. Each row is one unit.
