CUPED

Controlled-experiment Using Pre-Experiment Data — reduce metric variance using pre-experiment covariates.

When to use: You ran an A/B test and you have pre-experiment values of the same metric for the same users. CUPED subtracts the portion of variance explained by pre-period behaviour, making your test more sensitive without collecting more data.

Data structure: Four arrays of equal length for each group. Row N in post and row N in pre must refer to the same unit (user, store, market).

Example: You tested a new checkout flow on 5 stores for 4 weeks. You also have the 4 weeks before the experiment for those same stores. Enter post-experiment weekly revenue as post, pre-experiment weekly revenue as pre, for both groups.
