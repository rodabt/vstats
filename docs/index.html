VStats

Statistical & Machine Learning Library for V

A dependency-free library for linear algebra, statistics, probability distributions, machine learning, neural networks, and growth analytics.

Zero Dependencies Pure V Generic Types

Features

Linear Algebra

Vector and matrix operations with generic type support

Statistics

Descriptive statistics, hypothesis testing, ANOVA

Machine Learning

Regression, classification, clustering algorithms

Neural Networks

Deep learning layers, activations, loss functions

Growth Analytics

Funnel, cohort, attribution, and product metrics

Experimentation

A/B testing, CUPED, DiD, propensity score matching

Modules

linalg

Linear Algebra - Core vector and matrix operations with generic type support

stats

Descriptive Statistics & Hypothesis Testing - Measures of central tendency, dispersion, and statistical tests

growth

Growth & Product Analytics - Revenue metrics, funnel analysis, cohort analysis, and marketing attribution

ml

Machine Learning - Supervised and unsupervised learning algorithms

nn

Neural Networks - Deep learning layers, activations, and loss functions

prob

Probability Distributions - PDFs, CDFs, and inverse CDF for discrete and continuous distributions

hypothesis

Hypothesis Testing - Parametric and non-parametric tests for comparing distributions

optim

Optimization - Numerical differentiation and gradient descent primitives

Quick Start

import vstats.ml
import vstats.nn

fn main() {
    model := ml.linear_regression(x_train, y_train)
    predictions := ml.linear_predict(model, x_test)

    mut net := nn.sequential([10, 5, 1], "relu")
    net.train(x_train, y_train, nn.training_config(0.01, 100, 32))
}
