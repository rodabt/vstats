hypothesis

Statistical Hypothesis Testing — parametric and non-parametric tests for comparing distributions and assessing significance

Overview

import vstats.hypothesis // All tests return (statistic f64, p_value f64) // Reject H0 when p_value < alpha (default 0.05) t_stat, p := hypothesis.t_test_one_sample(data, 0.0, alpha: 0.05) chi2, p := hypothesis.chi_squared_test(contingency_table) w, p := hypothesis.wilcoxon_signed_rank_test(before, after) // Override significance level via TestParams tp := hypothesis.TestParams{ alpha: 0.01 } t_stat, p := hypothesis.t_test_two_sample(group_a, group_b, tp)

Every test returns (statistic f64, p_value f64). Reject the null hypothesis when p_value < alpha. The TestParams struct accepts alpha f64 = 0.05 and is optional — omit it to use the default.

API Reference

fn t_test_one_sample(x []f64, mu f64, tp TestParams) (f64, f64)

One-sample t-test. Tests whether the mean of sample x differs significantly from a known or hypothesized population mean mu.

ParameterTypeDescription
x[]f64Sample observations
muf64Hypothesized population mean (H₀: mean(x) = mu)
tpTestParamsOptional. alpha significance level, default 0.05

Returns (t_statistic f64, p_value f64) — two-tailed p-value under the t-distribution with n - 1 degrees of freedom.

fn t_test_two_sample(x []f64, y []f64, tp TestParams) (f64, f64)

Two-sample independent t-test (equal variances assumed). Tests whether the means of two independent samples differ significantly.

ParameterTypeDescription
x[]f64First sample
y[]f64Second sample
tpTestParamsOptional. alpha significance level, default 0.05

Returns (t_statistic f64, p_value f64) — two-tailed p-value under the t-distribution with pooled degrees of freedom (n₁ + n₂ - 2).

fn chi_squared_test(contingency [][]int) (f64, f64)

Chi-squared test of independence on a contingency table. Tests whether two categorical variables are independent of one another. Expected frequencies are computed automatically from marginal totals.

ParameterTypeDescription
contingency[][]intObserved frequency table; rows and columns represent categories of each variable

Returns (chi2_statistic f64, p_value f64) — p-value under the chi-squared distribution with (rows - 1) × (cols - 1) degrees of freedom.

fn chi_squared_gof_test(observed []f64, expected []f64) (f64, f64)

Chi-squared goodness-of-fit test. Tests whether an observed frequency distribution matches a specified theoretical distribution.

ParameterTypeDescription
observed[]f64Observed counts per category
expected[]f64Expected counts per category under H₀; must be same length as observed

Returns (chi2_statistic f64, p_value f64) — p-value under the chi-squared distribution with len(observed) - 1 degrees of freedom.

fn correlation_test(x []f64, y []f64, tp TestParams) (f64, f64)

Pearson correlation significance test. Tests whether the linear correlation between x and y differs significantly from zero (H₀: ρ = 0).

ParameterTypeDescription
x[]f64First variable
y[]f64Second variable; must be the same length as x
tpTestParamsOptional. alpha significance level, default 0.05

Returns (t_statistic f64, p_value f64) — two-tailed p-value using the t-approximation with n - 2 degrees of freedom.

fn wilcoxon_signed_rank_test(x []f64, y []f64) (f64, f64)

Wilcoxon signed-rank test. Non-parametric alternative to the paired t-test. Tests whether the distribution of pairwise differences x[i] - y[i] is symmetric about zero, without assuming normality.

ParameterTypeDescription
x[]f64First paired sample (e.g. pre-treatment measurements)
y[]f64Second paired sample (e.g. post-treatment measurements); must be the same length as x

Returns (W_statistic f64, p_value f64) — W is the sum of positive ranks. P-value is two-tailed using a normal approximation.

fn mann_whitney_u_test(x []f64, y []f64) (f64, f64)

Mann-Whitney U test. Non-parametric alternative to the two-sample independent t-test. Tests whether values in one sample tend to be larger than in the other, without assuming normality or equal variances.

ParameterTypeDescription
x[]f64First independent sample
y[]f64Second independent sample

Returns (U_statistic f64, p_value f64) — U counts the number of pairs where an observation from x exceeds one from y. P-value is two-tailed using a normal approximation.

fn shapiro_wilk_test(x []f64) (f64, f64)

Shapiro-Wilk normality test (approximation). Tests whether a sample could plausibly have been drawn from a normal distribution. Use this before applying parametric tests that assume normality.

ParameterTypeDescription
x[]f64Sample observations

Returns (W_statistic f64, p_value f64) — W is close to 1 for normally distributed data. A small p-value (p < 0.05) provides evidence against normality.

fn ks_test(sample1 []f64, sample2 []f64) (f64, f64)

Two-sample Kolmogorov-Smirnov test. Tests whether two samples come from the same underlying distribution. Distribution-free and sensitive to differences in shape, location, and scale.

ParameterTypeDescription
sample1[]f64First sample
sample2[]f64Second sample

Returns (D_statistic f64, p_value f64) — D is the maximum absolute difference between the two empirical CDFs. A large D (small p) suggests the distributions differ.

fn one_sample_t_test(x []f64, mu f64) (f64, f64)

Alias for t_test_one_sample using the default alpha of 0.05. Provided for backwards compatibility.

ParameterTypeDescription
x[]f64Sample observations
muf64Hypothesized population mean

Returns (t_statistic f64, p_value f64)

fn two_sample_t_test(x []f64, y []f64) (f64, f64)

Alias for t_test_two_sample using the default alpha of 0.05. Provided for backwards compatibility.

ParameterTypeDescription
x[]f64First sample
y[]f64Second sample

Returns (t_statistic f64, p_value f64)

Examples

One-sample t-test

import vstats.hypothesis

fn main() {
    // Test whether mean response time differs from 200ms
    times := [195.0, 202.0, 198.0, 210.0, 193.0, 207.0, 199.0]
    t_stat, p := hypothesis.t_test_one_sample(times, 200.0)

    println("t = ${t_stat:.4f},  p = ${p:.4f}")
    if p < 0.05 { println("Reject H0") } else { println("Fail to reject H0") }
}
Output: 
t = 0.2182,  p = 0.8347
Fail to reject H0

Chi-squared independence test

import vstats.hypothesis

fn main() {
    //            Success  Failure
    // Control:      45      55
    // Treatment:    60      40
    table := [[45, 55], [60, 40]]
    chi2, p := hypothesis.chi_squared_test(table)
    println("chi2 = ${chi2:.4f},  p = ${p:.4f}")
}
Output: 
chi2 = 4.5455,  p = 0.0330

Normality check then choose test

import vstats.hypothesis

fn main() {
    before := [12.0, 14.0, 11.0, 15.0, 10.0, 13.0]
    after  := [14.0, 17.0, 12.0, 18.0, 11.0, 16.0]

    w, p_norm := hypothesis.shapiro_wilk_test(before)
    if p_norm < 0.05 {
        // Not normal — use non-parametric test
        w_stat, p := hypothesis.wilcoxon_signed_rank_test(before, after)
        println("Wilcoxon W=${w_stat:.1f}  p=${p:.4f}")
    } else {
        t_stat, p := hypothesis.t_test_two_sample(before, after)
        println("t=${t_stat:.4f}  p=${p:.4f}")
    }
}
Output: 
Wilcoxon W=0.0  p=0.0313
