Real-World Examples

Practical applications of VStats for data science and product analytics.

Data Science

Customer Churn Prediction

Predict which customers are likely to churn using logistic regression.

import vstats.{ml, utils, stats}

fn main() {
    dataset := utils.load_breast_cancer() or { return }
    model := ml.logistic_classifier(x_train, y_train, 1000, 0.01)
    predictions := ml.logistic_classifier_predict(model, x_test, 0.5)
    cm := ml.confusion_matrix(y_test, predictions)
    println("Accuracy: ${(cm.accuracy() * 100):.1f}%")
}

Sales Forecasting

Forecast future sales using polynomial regression.

import vstats.{ml, stats}

E-commerce Funnel Optimization

Analyze conversion funnels to identify drop-off points.

import vstats.growth

fn main() {
    funnel := growth.create_funnel(
        ["Visitor", "Add to Cart", "Purchase"],
        [150000, 45000, 4500]
    )
    worst := funnel.highest_drop_off()
    println("Biggest drop-off: ${worst.from_name}")
}

Experimentation

A/B Testing

Statistical analysis of experiment results.

import vstats.experiment

fn main() {
    result := experiment.abtest(control, treatment, experiment.ABTestConfig{alpha: 0.05})
    println("Lift: ${(result.relative_lift * 100):.1f}%")
    println("p-value: ${result.p_value:.4f}")
    println("Significant: ${result.significant}")
}

Anomaly Detection

Detect anomalies using statistical thresholds.

import vstats.{stats}

fn main() {
    baseline_mean := stats.mean(baseline_data)
    baseline_std := stats.standard_deviation(baseline_data)
    threshold := baseline_mean + 2 * baseline_std
}
